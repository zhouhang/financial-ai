## 一、业务概述

本规则用于从用户上传的**手工凭证 Excel** 中提取数据，按照 **AI自动化逻辑** 定义的处理流程，**填充到用户上传的已有 BI 报表中**：

1. **BI 费用明细表**（用户上传的 `BI费用明细表.xlsx`）— 已有数据不全，需将手工凭证费用科目数据追加到该表末尾
2. **BI 损益毛利明细表**（用户上传的 `BI损益毛利明细表.xlsx`）— 已有数据不全，需将手工凭证收入成本数据追加到该表末尾

**输入文件（均由用户通过前端上传）**：
- 手工凭证 Excel（必填）：提取数据源
- BI费用明细表 Excel（必填）：待填充的目标表
- BI损益毛利明细表 Excel（必填）：待填充的目标表

### 2.2 摘要字段解析规则

摘要格式：`调整类型&客户+供应商+商户号(客户)+商户号(供应商)+税率`

```
摘要示例：调整收入&腾讯科技+武汉福禄网络科技有限公司++M001+6%
         ↓         ↓       ↓                              ↓   ↓
      调整类型  客户名称  供应商名称                    商户号  税率
```

- **`&` 分隔**：`&` 左侧为**调整类型**（调整收入 / 调整成本 / 调整费用），右侧为 `+` 拼接的明细
- **`+` 分隔**（顺序固定）：客户、供应商、商户号(客户)、商户号(供应商)、税率
  - 若某字段为空，则对应位置空字符串，`+` 数量不变
  - 费用类摘要中 `+` 右侧**仅有税率**（若为空则税率=0）

## 四、处理流程一：手工凭证 → 手工收入成本明细表（章节一）

### 4.1 科目过滤

取科目名称包含以下内容的凭证行：
- `主营业务收入`（含所有子科目）
- `主营业务成本_销售成本`（或 `主营业务成本-销售成本`）
- `主营业务成本_技术服务成本`（或 `主营业务成本-技术服务成本`）

### 4.2 核算类型判断

- 科目名称为 `主营业务收入_寄售收入` → 核算类型 = **"净额"**
- 其他所有科目 → 核算类型 = **"总额"**

### 4.3 税率推断

优先从**科目名称后缀**推断税率：

| 科目名称后缀 | 税率 |
|-------------|------|
| 以 `-13%` 或 `_13%` 结尾 | 13% |
| 以 `-9%` 或 `_9%` 结尾 | 9% |
| 以 `-3%` 或 `_3%` 结尾 | 3% |
| 以 `-1%` 或 `_1%` 结尾 | 1% |
| 其他科目 | **6%**（默认） |

若摘要 `+` 解析出税率字段（如 `6%`、`13%`），则以摘要中的税率优先。

### 4.4 eas 收入/成本不含税计算

| 条件 | eas收入不含税 | eas成本不含税 |
|------|-------------|-------------|
| 调整类型=调整收入 且 科目=主营业务收入_寄售收入 | = 贷方发生额 | = 0 |
| 调整类型=调整成本 且 科目=主营业务收入_寄售收入 | = 0 | = 负数(贷方发生额) |
| 科目=主营业务收入_销售收入 或 主营业务收入_技术服务收入 | = 贷方发生额 | = 0 |
| 科目=主营业务成本_销售成本 或 主营业务成本_技术服务成本 | = 0 | = 借方发生额 |

### 4.5 含税金额计算

```
含税销售额       = eas收入不含税 × (1 + 税率)
含税采购成本     = eas成本不含税 × (1 + 税率)
含税差额收入     = 含税销售额 - 含税采购成本
```

## 五、处理流程四：手工凭证 → 手工费用明细表（章节四）

### 5.1 费用科目过滤

取科目名称包含以下内容的凭证行（管理费用需剔除研发费子科目）：

```
研发支出_费用化支出
主营业务成本_直接服务费
主营业务成本_工资性支出
其他业务收入
其他业务成本
管理费用（剔除"管理费用_研发费"或"管理费用-研发费"子科目）
销售费用
财务费用
营业外收入
营业外支出
投资收益
资产减值损失
所得税
```

### 5.2 费用摘要解析

- 摘要按 `&` 分列 → **调整类型** = "调整费用"
- 摘要按 `+` 分列 → **税率**（若为空则税率=0%）

### 5.3 费用科目三级分列

按科目名称的 `_` 拆分：
- **eas一级科目** = 第1段
- **费用二级项目** = 第2段（无则空）
- **费用三级项目** = 第3段（无则空）

### 5.4 费用金额计算

```
eas不含税金额 = 借方发生额 + 贷方发生额
eas含税金额   = eas不含税金额 × (1 + 税率)
eas税额       = eas含税金额 - eas不含税金额
```

## 九、注意事项

1. **金额精度**：所有金额保留 2 位小数，使用 `round(value, 2)` 处理
2. **空值处理**：借方/贷方发生额为空时视为 0
3. **科目识别**：优先通过科目名称文本匹配，其次通过科目编码前缀
4. **摘要解析**：`&` 仅取第一个分隔符左侧为调整类型，右侧再按 `+` 解析
5. **税率解析**：税率字符串如 `6%` 需转换为小数 `0.06`
6. **关联判断**：客户和供应商分别独立判断，均需与公司主体列表比对
