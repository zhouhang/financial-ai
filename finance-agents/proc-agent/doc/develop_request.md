# 开发需求
基于当前项目，我们正在开发一个数据收集、核对、整理，分析的数据分析agent；

## 开发详细需求
1. 我们希望我们新开发的数据整理的agent(data-process),中文名字叫审计整理数字员工；用户可以在这个agent下定义针对不同业务的多个skill定义用于完成不同的业务场景处理；当前agent可以调用和使用的skill文件在skills目录下存放；
2. 每个skill均有一个描述该skill功能的md文件，md文件中有描述该skill的功能和取数规则的说明;
3. 所有原始数据文件均存储在data目录下,文件可能有excel或pdf，图片等格式的文件;
4. agent可以读取skills目录下的skill文件列表，然后根据文件的内容描述，生成对应的python处理脚本，并运行脚本进行功能逻辑处理；
5. 所有生成脚本存放在scripts目录下；
6. 所有生成的最终结果文件存放到result目录下；

请根据上面的需求做一个该agent开发的规划，并生成对应技术架构图，业务功能流程图等。
