# 业务流程：
1. 首先我们需要生成BI损益毛利明细表(手工类)和BI费用明细表(手工类)，其中这2个表的数据由手工凭证表同步而来；
2. 根据同步规则我们从手工凭证表里过滤和获取对应的记录数据，然后生成相应的BI损益毛利明细表(手工类)和BI费用明细表(手工类)
3. 生成后将BI损益毛利明细表(手工类)和BI费用明细表(手工类)与原来的BI损益毛利明细表和BI费用明细表合并;
4. 然后跟从金蝶下载的数据进行比对，找到差异后，调整手工类的表数据；
5. 然后将手工类的表上传数据中台，刷新供应商毛利表；
6. 然后导出供应商毛利表，再手工补充没有的其它类别的数据
7. 根据上述文件出分析报告
