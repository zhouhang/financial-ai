## 一、业务概述

本规则用于对企业库存商品数据进行分析，识别存货周转、库龄结构、呆滞商品等情况，生成库存分析报告。

## 五、分析规则

### 5.1 存货周转率计算

```python
存货周转率 = 营业成本 / 平均存货余额
平均存货余额 = (期初存货 + 期末存货) / 2
存货周转天数 = 365 / 存货周转率
```

### 5.2 呆滞商品识别

**判定条件**（满足任一即可）:

| 条件 | 判定标准 |
|------|----------|
| 库龄 | 库龄 > 730 天 |
| 周转率 | 近 1 年无出库记录 |
| 数量 | 库存数量 > 年销售量×2 |

### 5.3 临期商品识别

**适用商品**: 有保质期要求的商品

**判定条件**:
```python
剩余保质期 = 保质期截止日 - 当前日期
if 剩余保质期 <= 90 天:
    标记 = "临期"
```

### 5.4 ABC 分类分析

**分类标准**:

| 分类 | 标准 | 管理策略 |
|------|------|----------|
| A 类 | 金额占比前 70% 的商品 | 重点管理，精确控制 |
| B 类 | 金额占比 70%-90% 的商品 | 常规管理 |
| C 类 | 金额占比后 10% 的商品 | 简化管理 |

## 六、汇总统计规则

### 6.1 按库龄汇总

```python
按库龄分类分组：
- 各类别商品数量 = count(*)
- 各类别商品金额 = sum(金额)
- 各类别占比 = 各类别金额 / 总金额
```

### 6.2 按商品类别汇总

```python
按商品类别分组：
- 类别库存金额 = sum(金额)
- 类别周转率 = 类别销售成本 / 类别平均库存
```

### 6.3 库龄结构分析

| 库龄分类 | 商品数 | 金额 | 占比 |
|----------|--------|------|------|
| 1 年以内 | X | A | A% |
| 1-2 年 | Y | B | B% |
| 2-3 年 | Z | C | C% |
| 3 年以上 | W | D | D% |

## 八、校验规则

1. **金额平衡**: ∑(数量×单价) = 总金额
2. **数量平衡**: 期初 + 入库 - 出库 = 期末
3. **分类完整**: A+B+C 类金额合计 = 总库存金额

## 十、注意事项

### 10.1 特殊商品处理

1. **季节性商品**: 考虑季节性因素，不单纯以库龄判断
2. **备品备件**: 某些备件虽库龄长但为必要储备
3. **定制商品**: 专用材料可能周转较慢

### 10.2 单价处理

- 优先使用加权平均单价
- 如库存表已有单价，直接使用
- 注意单价是否含税
