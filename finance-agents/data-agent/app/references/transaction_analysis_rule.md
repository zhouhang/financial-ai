## 五、分析规则

### 5.1 大额交易识别

**阈值配置**:
- 大额收入阈值：≥ 100,000 元
- 大额支出阈值：≥ 100,000 元

**标记规则**:
```python
if 交易金额 >= 100000:
    标记 = "大额交易"
```

### 5.2 频繁交易识别

**判定规则**:
- 同一对方账户，单日交易次数 ≥ 5 次

**标记规则**:
```python
按交易日期 + 对方账号分组
if 组内交易次数 >= 5:
    标记 = "频繁交易"
```

### 5.3 异常交易识别

**异常类型**:

| 异常类型 | 判定条件 |
|----------|----------|
| 整数大额 | 交易金额为整数且≥ 50,000 |
| 夜间交易 | 交易时间在 22:00-06:00 |
| 摘要为空 | 摘要字段为空或"无" |
| 快进快出 | 同日同金额进出 |

## 六、汇总统计规则

### 6.1 按日汇总

```python
按交易日期分组：
- 日收入合计 = sum(借方金额)
- 日支出合计 = sum(贷方金额)
- 日净流量 = 日收入合计 - 日支出合计
- 日交易笔数 = count(*)
```

### 6.2 按月汇总

```python
按月份分组：
- 月收入合计 = sum(借方金额)
- 月支出合计 = sum(贷方金额)
- 月净流量 = 月收入合计 - 月支出合计
```

### 6.3 按交易对方汇总

```python
按对方账号分组：
- 交易总笔数 = count(*)
- 交易总金额 = sum(交易金额)
- 首次交易日期 = min(交易日期)
- 末次交易日期 = max(交易日期)
```

## 八、校验规则

1. **余额连续性校验**: 上期余额 + 本期收入 - 本期支出 = 本期余额
2. **借贷平衡校验**: 期初余额 + ∑借方 - ∑贷方 = 期末余额
3. **交易笔数校验**: 明细笔数 = 汇总笔数
