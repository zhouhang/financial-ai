## 🎓 学习资源

### 相关代码示例

**如何使用新的匹配函数**：
```python
from app.graphs.reconciliation import _find_matching_items

# 查找与"删除订单号"相匹配的配置项
config_items = [
    {"description": "订单号分组配置"},
    {"description": "金额转换规则"},
]

matches = _find_matching_items("删除订单号", config_items, threshold=0.5)
# → [0]  # 匹配了第一个配置项
```

### 扩展功能

**如果需要调整匹配严格程度**：
```python
# 更宽松的匹配
matches = _find_matching_items(target, items, threshold=0.3)

# 更严格的匹配
matches = _find_matching_items(target, items, threshold=0.7)
```
