## Why

当前文件分析结果以表格形式展示时，存在两个用户体验问题：(1) 文件名较长时导致表格列错位或内容溢出；(2) 当字段（列）较多时，表格整体过宽，难以在有限屏幕空间内完整浏览。现在需要优化表格展示方案，提升可读性和可用性。

## What Changes

- 设计响应式表格布局，支持横向滚动和列自适应
- 实现文件名过长时的省略/截断策略，支持悬停 tooltip 显示完整名称
- 添加列可见性控制，允许用户自定义显示哪些字段
- 实现表格列宽拖拽调整功能
- 增加紧凑/标准/展开三种视图模式切换

## Capabilities

### New Capabilities

- `responsive-table`: 响应式表格组件，支持横向滚动、列自适应、列可视控制、视图模式切换
- `filename-ellipsis`: 文件名省略策略，支持截断显示和 tooltip 展开

### Modified Capabilities

- (无)

## Impact

- 前端组件：`finance-web/src/components/` 目录下的文件分析相关组件
- 可能需要新增通用表格组件，复用于其他模块
- 无后端 API 变更
- 无数据库变更
